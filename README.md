# Surplux - Create Account

A simple and responsive **Sign Up** webpage built using **HTML, CSS, and JavaScript**. The project includes password strength checking and password confirmation validation with a modern glassmorphism design.

## FEATURES

* User registration form
* Password strength checker (Weak, Medium, Strong)
* Confirm password validation
* Glassmorphism user interface
* Responsive design
* Success message after account creation

## TECH STACK

* HTML
* CSS
* JavaScript

## WORKING

**User Input** – The user enters a username, email, password, and confirm password.

**Password Strength** – The application checks the password strength based on:

* Minimum 8 characters
* Uppercase letter
* Lowercase letter
* Number
* Special character

**Validation** – The application verifies that the password and confirm password match.

**Account Creation** – If all validations are successful, a success message is displayed. If the passwords do not match, an error message is shown.
